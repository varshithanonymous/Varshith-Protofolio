Research Work

SRC 2025, Zayed University, Dubai
IoT-Based Smart Umbrella System for Outdoor Vehicle Protection using Machine Learning

This research presents a compact and intelligent IoT-based Smart Umbrella System powered by an ESP-32 microcontroller for outdoor vehicle protection. Unlike traditional reactive or hydraulic systems, this design utilizes an integrated network of sensors (YL-83, VEML6030, and NEO-6M GPS) for dynamic weather analysis.

A Decision Tree-based ML model predicts environmental triggers like rainfall or high UV radiation, minimizing false activations. The system automatically deploys a servo-driven umbrella and notifies users via an Android app, ensuring real-time response with optimal energy efficiency.

Tech Stack: ESP-32, IoT, Decision Tree, Android App, YL-83, VEML6030, GPS

SRC 2025, Zayed University, Dubai
Real-Time Vision-and-Voice-Based Navigation System using Deep Learning and LLMs

This project integrates YOLOv10 object detection and LLM-based audio feedback to provide intelligent navigation assistance for both visually impaired users and low-visibility driving environments.

It employs spatio-temporal motion tracking (MOG2) with sensor fusion using LiDAR and IMU data, generating real-time contextual voice prompts via Google TTS and eSpeak. The system runs on an ESP32 microcontroller, enabling wireless communication, compact design, and energy-efficient operation.

Tech Stack: YOLOv10, LLMs, Flask, ESP32, LiDAR, IMU, Python
